Workshop de Python na FASE (Olinda) - Divulgação dos Cursos Presenciais

Publicado em: Seg 26 maio 2014. | Por: Gileno Filho | Arquivado em: cursos

Olá pessoal, no sábado passado (24/05/2014), eu e Péricles Miranda demos um workshop de Python na Faculdade Santa Emília - FASE. Mostramos uma visão geral sobre Python e o seu ecossistema, após a apresentação começamos a mostrar exemplos de código em Python e demonstrando o seu poder em diversos ambientes: Desktop, Web, Computação Científica ...

Esse workshop faz parte de uma sério de workshop's que iremos fazer em Recife e região metropolitana para divulgar os cursos presencias que irão ocorrer em Julho, após a Copa do Mundo de Futebol.

  • Python para Desenvolvedores
  • Python na Web com Django

O código está no github: https://github.com/gileno/workshop-fase
